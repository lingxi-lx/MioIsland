Pixel Character — Asset Sources

Existing free assets vs custom-made, pick a direction

A

Use Free Sprite Sheets (CC0)

From itch.io / OpenGameArt. Thousands of free 16x16 / 32x32 animated characters with idle, walk, attack states. CC0 license, directly usable in open source product.

Pros

  • Immediately available, professional quality
  • Multiple characters to choose from
  • Already animated with sprite sheets
  • CC0 = no licensing issues

Cons

  • May not have "typing at computer" state
  • Generic game characters, not coding-themed
  • Need to adapt idle/walk/attack to our states
B

AI Generate Custom Sprites

Use PixelLab or PixelBox to generate a custom coding-themed character with exactly the states we need.

Pros

  • Perfectly themed — coder at desk, laptop, etc.
  • Custom states: typing, thinking, celebrating
  • Unique brand identity for CodeIsland

Cons

  • Quality varies, may need manual cleanup
  • Need to verify licensing for open source
C

Hand-draw in Piskel / Aseprite

Use Piskel (free, browser-based) or Aseprite to hand-draw a minimalist character with exactly the right vibe.

Pros

  • Full creative control
  • Can be extremely minimal (8x8 or 12x12)
  • 100% original, no licensing concerns
  • Exports directly as sprite sheet PNG

Cons

  • Requires pixel art skill or iteration
  • Most time-consuming option
D

Programmatic Canvas (current v2 approach, refined)

Keep the current approach — draw characters in code with Canvas/SwiftUI. Refine the animations we just made to be more expressive.

Pros

  • No external assets needed, pure code
  • Infinitely tweakable at runtime
  • Can react dynamically to real data (e.g., token count affects animation speed)
  • Smallest bundle size

Cons

  • Harder to get "professionally drawn" feel
  • Animation quality depends on code, not art
